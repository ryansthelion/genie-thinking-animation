# Genie — Thinking & Trace Prototype

Interactive prototype of the **Animation 1** thinking flow from [Genie Unified Components (Figma)](https://www.figma.com/design/st5IOjhahvmvXyDbjvMpHB/%F0%9F%A7%B0-Genie--Unified-Components-?node-id=9982-71369).

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Animation steps

The prototype auto-advances through the Figma frames in section **Animation 1**:

1. User message only
2. Genie lamp thinking animation (linked frame `9982:71369`)
3. Collapsed “Thinking” drawer with shimmer
4. Thinking trace — pulsing status dot
5. Step title: “Understanding the question”
6. Step description
7. Tool call with search results

Use **Previous / Next / Play / Pause** to explore manually.
